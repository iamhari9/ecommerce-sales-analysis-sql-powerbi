# Dataset Information

Due to GitHub file size limitations, the cleaned dataset is not included in this repository.

Data Source:
- Online Retail Dataset (UCI Machine Learning Repository)
https://archive.ics.uci.edu/ml/datasets/online+retail

Data Preparation:
- Raw data was imported into SQL Server
- Data cleaning and transformation were performed using SQL
- All data preparation steps are documented in:
  - sql/data_cleaning.sql

Reproducibility:
Anyone can reproduce the analysis by downloading the dataset from the source
and running the SQL scripts provided in this repository.

A small sample dataset is included for reference purposes only.
